# HomeAssistant y portainer Docker Compose

## Deployment
```bash
git clone https://github.com/alejandro14972/DockerHomeAssistant.git
cd DockerHomeAssistant
mkdir -p ~/homeassistant_config
mkdir -p ~/portainer_data
docker-compose up -d
```
## To open HomeAssistant 
```bash
http://localhost:8123
````
